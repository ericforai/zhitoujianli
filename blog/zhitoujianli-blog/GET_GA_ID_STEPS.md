# 🎯 从Google Analytics界面获取测量ID

## 📋 详细操作步骤

### 第一步：点击"管理"按钮

- 在您当前的GA4界面左下角，点击**齿轮图标**（管理）
- 或者点击页面底部的**"管理"**卡片

### 第二步：选择数据流

- 在管理页面中，找到**"数据流"**部分
- 点击您网站的数据流（名称类似"网站"或您的网站名称）

### 第三步：复制测量ID

- 在数据流详情页面顶部，找到**"测量ID"**
- 复制以`G-`开头的ID（例如：`G-XXXXXXXXXX`）

## 🚀 快速设置到博客

获取到测量ID后，运行以下命令：

```bash
cd /root/zhitoujianli/blog/zhitoujianli-blog
./setup-ga-quick.sh
```

然后输入您复制的测量ID即可。

## 📊 验证设置

设置完成后：

1. 访问您的博客网站
2. 在Google Analytics的"实时"报告中查看数据
3. 使用浏览器开发者工具检查网络请求

## 🔧 如果遇到问题

- **找不到数据流**：需要先创建网站数据流
- **测量ID格式错误**：确保以`G-`开头
- **数据不显示**：等待5-10分钟，GA数据有延迟
