## 🎯 目标用户群体概览

智投简历的核心用户是**正在求职或计划求职的个人**，包括：

1. **应届毕业生**（22-24岁）
2. **职场跳槽者**（25-35岁）
3. **转行求职者**（25-40岁）

---

## 👤 用户画像 1：应届毕业生

### 基本信息

| 属性 | 描述 |
|------|------|
| **年龄** | 22-24岁 |
| **教育背景** | 本科/硕士应届毕业生 |
| **职业状态** | 即将毕业，首次求职 |
| **所在城市** | 一二线城市（北上广深杭等） |
| **月收入期望** | 6K-15K（本科），10K-25K（硕士） |

### 核心痛点

1. **简历经验不足**
   - 只有校园项目/实习经历
   - 不知道如何包装亮点
   - 担心简历太"水"

2. **求职方向迷茫**
   - 不知道自己适合什么岗位
   - 海投各类职位，缺乏针对性
   - 面对大量JD感到无从下手

3. **投递效率低下**
   - 每天花2-3小时手动投递
   - 简历投递后回复率极低（<2%）
   - 不知道如何优化投递策略

4. **沟通能力欠缺**
   - 打招呼语千篇一律
   - 不知道如何吸引HR注意
   - 面对"自我介绍"不知所措

5. **心理压力大**
   - 求职焦虑、自我怀疑
   - 同学都拿到Offer，自己还在投递
   - 父母催促，压力倍增

### 行为特征

- **求职方式**：主要通过Boss直聘、智联招聘、拉勾网
- **信息渠道**：知乎、小红书、B站、豆瓣小组
- **决策因素**：看重口碑评价、同学推荐、免费试用
- **活跃时间**：晚上20:00-24:00（白天可能有课/实习）
- **社交习惯**：喜欢在社群中交流、分享经验

### 使用场景

1. **校招高峰期**（9-11月、3-5月）
   - 大量投递校招岗位
   - 需要快速筛选合适机会
   - 希望提高简历通过率

2. **实习转正**（6-8月）
   - 在实习和校招之间权衡
   - 需要同时准备多个方向
   - 时间紧迫，需要高效工具

3. **延迟毕业/二战求职**（全年）
   - 压力更大，更需要效率
   - 希望快速找到合适工作
   - 对工具依赖度更高

### 核心需求

| 需求类型 | 具体需求 | 优先级 |
|---------|---------|--------|
| **效率提升** | 减少手动投递时间，自动匹配岗位 | ⭐⭐⭐⭐⭐ |
| **简历优化** | 如何包装校园经历，提升竞争力 | ⭐⭐⭐⭐⭐ |
| **精准匹配** | 找到真正适合的岗位，不再盲目海投 | ⭐⭐⭐⭐ |
| **沟通指导** | 生成专业打招呼语，提高回复率 | ⭐⭐⭐⭐ |
| **心理支持** | 数据可视化，减少焦虑感 | ⭐⭐⭐ |

### 价值主张

**对应届生的核心价值**：

> "智投简历让你的第一次求职不再迷茫。AI帮你找到最匹配的岗位，生成专业的打招呼语，投递效率提升10倍。把时间花在准备面试上，而不是海投简历。"

### 营销触点

1. **知乎**：回答"应届生求职"相关问题
2. **小红书**：发布"校招攻略""简历优化"内容
3. **B站**：制作"求职Vlog""应届生逆袭"视频
4. **豆瓣**：在求职小组分享经验
5. **高校社群**：通过学长学姐推荐

### 转化路径

```
痛点认知（知乎/小红书）
↓
了解产品（官网/公众号）
↓
免费试用（注册账号）
↓
体验效果（投递5-10个岗位）
↓
付费转化（拿到面试邀请后）
```

## 👔 用户画像 2：职场跳槽者

### 基本信息

| 属性 | 描述 |
|------|------|
| **年龄** | 25-35岁 |
| **工作经验** | 3-8年 |
| **职业状态** | 在职，计划跳槽 |
| **所在城市** | 一二线城市 |
| **月收入期望** | 15K-50K（晋升或加薪30%-50%） |

### 核心痛点

1. **时间有限**
   - 白天要上班，只能晚上/周末投递
   - 希望高效利用碎片时间
   - 不希望被现公司发现跳槽意向

2. **精准匹配困难**
   - 不想"向下兼容"（降薪/降职）
   - 需要找到真正匹配的高质量岗位
   - 对公司文化、团队氛围有要求

3. **沟通效率低**
   - HR太多，回复慢
   - 不想浪费时间在不合适的岗位上
   - 希望快速进入面试环节

4. **简历更新麻烦**
   - 工作忙，没时间频繁修改简历
   - 不同岗位需要不同简历版本
   - 希望一次解析，多次使用

5. **信息不对称**
   - 不知道市场行情（薪资、岗位需求）
   - 不清楚自己的市场价值
   - 担心被"画饼"

### 行为特征

- **求职方式**：Boss直聘、拉勾网、猎聘、LinkedIn
- **信息渠道**：知乎、脉脉、行业社群、公众号
- **决策因素**：看重效率、数据、专业度、安全性
- **活跃时间**：晚上21:00-23:00，周末全天
- **社交习惯**：注重隐私，不愿公开讨论跳槽

### 使用场景

1. **主动跳槽**（年后3-5月，年中9-10月）
   - 对现状不满，主动寻找机会
   - 有明确的职业规划
   - 希望快速高效

2. **被动求职**（全年）
   - 公司经营不善/裁员风险
   - 被猎头推荐好机会
   - 希望低调进行

3. **职业转型**（全年）
   - 从技术转管理
   - 从传统行业转互联网
   - 需要精准定位

### 核心需求

| 需求类型 | 具体需求 | 优先级 |
|---------|---------|--------|
| **效率最大化** | 用最少时间找到最合适的机会 | ⭐⭐⭐⭐⭐ |
| **精准匹配** | 只看高质量岗位，避免浪费时间 | ⭐⭐⭐⭐⭐ |
| **隐私保护** | 不被现公司发现跳槽意向 | ⭐⭐⭐⭐ |
| **专业沟通** | 打招呼语体现专业度，快速进入面试 | ⭐⭐⭐⭐ |
| **数据分析** | 了解市场行情，薪资对比 | ⭐⭐⭐ |

### 价值主张

**对跳槽者的核心价值**：

> "智投简历让你在职跳槽更高效、更安全。AI自动筛选高匹配度岗位（≥75%），生成专业打招呼语，投递效率提升10倍。把有限的时间用在真正有价值的机会上。"

### 营销触点

1. **知乎**：回答"跳槽攻略""职场晋升"问题
2. **脉脉**：分享"跳槽经验""薪资对比"
3. **公众号**：推送"跳槽指南""行业分析"
4. **LinkedIn**：建立专业形象
5. **行业社群**：口碑传播

### 转化路径

```
效率痛点（时间有限）
↓
了解工具（知乎/公众号）
↓
免费试用（注册账号）
↓
体验效果（精准匹配+高回复率）
↓
付费转化（拿到2-3个高质量Offer后）
```

## 🔄 用户画像 3：转行求职者

### 基本信息

| 属性 | 描述 |
|------|------|
| **年龄** | 25-40岁 |
| **工作经验** | 3-15年（跨行业） |
| **职业状态** | 在职或离职gap中 |
| **所在城市** | 一二线城市 |
| **月收入期望** | 10K-30K（可能降薪20%-30%） |

### 核心痛点

1. **简历缺乏亮点**
   - 过往经验与目标岗位不直接相关
   - 不知道如何包装"可迁移技能"
   - HR第一眼就可能Pass

2. **投递策略困难**
   - 不知道应该投什么类型的岗位
   - 担心被认为"不专业"
   - 海投效果差，打击信心

3. **沟通难度大**
   - 如何解释转行原因
   - 如何证明自己能胜任
   - 打招呼语难以打动HR

4. **心理压力巨大**
   - 年龄焦虑（担心年龄大了转行难）
   - 经济压力（可能降薪或gap期）
   - 自我怀疑（是否真的适合转行）

5. **信息获取困难**
   - 不了解目标行业的真实需求
   - 不知道自己的优势在哪里
   - 缺乏行业内人脉指导

### 行为特征

- **求职方式**：Boss直聘、拉勾网、智联招聘
- **信息渠道**：知乎、公众号、行业社群、线下活动
- **决策因素**：看重成功案例、专业指导、长期价值
- **活跃时间**：全天（如果离职gap中）
- **社交习惯**：寻求认同、交流经验、抱团取暖

### 使用场景

1. **主动转行**（全年）
   - 对现行业/岗位不满
   - 追求职业发展或兴趣
   - 有明确目标行业

2. **被动转行**（全年）
   - 原行业衰退/裁员
   - 被迫寻找新方向
   - 压力更大

3. **探索阶段**（全年）
   - 还在犹豫是否转行
   - 希望了解目标行业机会
   - 边工作边投递

### 核心需求

| 需求类型 | 具体需求 | 优先级 |
|---------|---------|--------|
| **简历包装** | 如何包装跨行业经验，提炼可迁移技能 | ⭐⭐⭐⭐⭐ |
| **精准定位** | 找到适合转行的"过渡岗位" | ⭐⭐⭐⭐⭐ |
| **专业沟通** | 打招呼语解释转行原因，建立信任 | ⭐⭐⭐⭐ |
| **信心建立** | 成功案例激励，减少焦虑 | ⭐⭐⭐⭐ |
| **行业洞察** | 了解目标行业需求和趋势 | ⭐⭐⭐ |

### 价值主张

**对转行者的核心价值**：

> "智投简历帮你发现跨行业的核心竞争力。AI自动提取可迁移技能，匹配适合转行的岗位，生成专业的打招呼语。转行不再是劣势，而是多元经验的优势。"

### 营销触点

1. **知乎**：回答"转行求职""职业转型"问题
2. **公众号**：推送"转行成功案例""可迁移技能分析"
3. **小红书**：分享"转行经验""避坑指南"
4. **线下活动**：职业转型沙龙、行业交流会
5. **社群**：转行求职互助群

### 转化路径

```
转行困惑（简历不匹配）
↓
看到成功案例（知乎/公众号）
↓
了解产品（如何帮助转行）
↓
免费试用（看到可迁移技能）
↓
体验效果（匹配到过渡岗位）
↓
付费转化（拿到第一个转行Offer后）
```

## 🚀 用户获取优先级

### 第一阶段（Month 1-2）：聚焦应届生

**原因**：
- 痛点明确，需求强烈
- 社交活跃，传播能力强
- 决策快，转化周期短
- 适合冷启动

**目标**：获取500名应届生用户

---

### 第二阶段（Month 3-4）：扩展跳槽者

**原因**：
- 付费意愿强，LTV高
- 专业度要求高，建立信任后忠诚度高
- 适合品牌升级阶段

**目标**：获取300名跳槽者用户

---

### 第三阶段（Month 5-6）：覆盖转行者

**原因**：
- 需求复杂，需要更强产品能力
- 案例积累后更容易说服
- 适合产品成熟阶段

**目标**：获取200名转行者用户

---

## 📝 总结

**核心用户群体**：求职者（应届生、跳槽者、转行者）

**共同痛点**：
1. 投递效率低
2. 匹配度不准
3. 沟通能力弱
4. 缺乏数据追踪

**智投简历的核心价值**：
1. AI自动匹配岗位（精准度）
2. 生成个性化打招呼语（专业度）
3. 投递记录追踪（数据化）
4. 节省时间（效率）

**营销策略**：
- 分阶段获取不同用户群体
- 内容+案例+数据驱动
- 口碑传播+社群运营
- 免费试用+付费转化
