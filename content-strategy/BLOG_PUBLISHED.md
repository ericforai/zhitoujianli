## 📞 技术支持

### 博客文件位置

```bash
# 线上部署路径
/var/www/zhitoujianli/build/blog/

# 源代码路径
/root/zhitoujianli/frontend/public/blog/

# 内容策略文档
/root/zhitoujianli/content-strategy/
```

### 如何修改博客内容

1. **编辑源文件**：

```bash
vi /root/zhitoujianli/frontend/public/blog/2025-job-market-truth.html
```

2. **重新部署**：

```bash
cd /root/zhitoujianli
echo "YES" | ./deploy-frontend.sh
```

3. **清除浏览器缓存**：`Ctrl + Shift + R`

### 如何添加新文章

1. **创建HTML文件**：

```bash
vi /root/zhitoujianli/frontend/public/blog/new-article.html
```

2. **更新博客列表页**：

```bash
vi /root/zhitoujianli/frontend/public/blog/index.html
```

（在文章列表中添加新卡片）

3. **更新Sitemap**：

```bash
vi /root/zhitoujianli/frontend/public/blog/sitemap-0.xml
```

（添加新URL）

4. **重新部署**：

```bash
echo "YES" | /root/zhitoujianli/deploy-frontend.sh
```
