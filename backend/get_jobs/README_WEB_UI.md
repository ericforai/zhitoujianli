# AI找工作助手 - Web可视化界面

## 🎯 功能特点

- **可视化配置**: 通过Web界面配置所有参数，无需手动编辑YAML文件
- **实时监控**: 实时显示程序运行状态和投递统计
- **日志查看**: 实时查看程序运行日志
- **多平台支持**: 支持Boss直聘、前程无忧、拉勾网、猎聘网、智联招聘
- **一键启动**: 通过Web界面一键启动/停止程序

## 🚀 快速开始

### 1. 启动Web界面

```bash
# 方法1: 使用启动脚本（推荐）
./start_web_ui.sh

# 方法2: 手动启动
pip3 install flask pyyaml
python3 simple_web.py
```

### 2. 访问Web界面

打开浏览器访问: http://localhost:8080

## 📋 使用说明

### 配置参数

#### Boss直聘配置
- **搜索关键词**: 输入要搜索的职位关键词，用逗号分隔
- **工作城市**: 选择工作城市（可多选）
- **工作经验**: 选择工作经验要求（可多选）
- **薪资要求**: 选择薪资范围
- **期望薪资**: 设置期望薪资范围（K为单位）
- **投递间隔**: 设置每次投递的间隔时间（秒）
- **过滤选项**: 是否过滤不活跃HR
- **AI功能**: 是否启用AI智能打招呼
- **图片简历**: 是否发送图片简历
- **打招呼语**: 自定义打招呼内容

#### AI配置
- **个人介绍**: 用于AI生成个性化打招呼语
- **AI提示词**: 自定义AI提示词模板
- **消息推送**: 是否启用企业微信消息推送

### 操作流程

1. **配置参数**: 在"Boss直聘"和"AI配置"标签页中设置相关参数
2. **保存配置**: 点击"保存配置"按钮保存设置
3. **选择平台**: 在侧边栏选择要启动的平台
4. **启动程序**: 点击"启动程序"按钮开始投递
5. **监控状态**: 在"运行日志"标签页查看实时日志和投递统计
6. **停止程序**: 需要时点击"停止程序"按钮

### 状态监控

- **程序状态**: 显示当前程序运行状态（运行中/已停止）
- **投递统计**: 实时显示成功投递的岗位数量
- **运行日志**: 显示详细的程序运行日志
- **自动刷新**: 可开启/关闭日志自动刷新

## 🔧 技术架构

- **后端**: Python Flask + PyYAML
- **前端**: Bootstrap 5 + JavaScript
- **配置管理**: YAML文件读写
- **进程管理**: subprocess + threading
- **日志监控**: 实时日志流处理

## 🛠️ 故障排除

### 常见问题

1. **端口占用**: 如果8080端口被占用，可以修改`simple_web.py`中的端口号
2. **依赖缺失**: 确保安装了Python3和pip3，然后运行`pip3 install flask pyyaml`
3. **权限问题**: 确保启动脚本有执行权限`chmod +x start_web_ui.sh`
4. **Java环境**: 确保系统已安装Java 21和Maven

### 日志查看

- **Web界面**: 在"运行日志"标签页查看实时日志
- **文件日志**: 查看`logs/`目录下的日志文件
- **控制台**: 查看启动Web服务的控制台输出

## 🎉 使用建议

1. **首次使用**: 建议先配置好所有参数，保存后再启动程序
2. **参数调整**: 可以随时修改配置并保存，但需要重启程序才能生效
3. **监控投递**: 建议定期查看投递统计和日志，确保程序正常运行
4. **反检测**: 程序已内置反检测机制，建议保持默认设置
