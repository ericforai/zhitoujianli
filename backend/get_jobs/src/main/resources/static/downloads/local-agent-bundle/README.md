# 智投简历 - 本地Agent快速开始

## 3步开始投递（与网页一致）

1. 双击启动器  
   - Mac/Linux: `start.command`  
   - Windows: `start.bat`

2. 首次按提示登录智投简历账号，并在浏览器里扫码登录Boss直聘

3. 回到网站 Dashboard，点击「开始投递」

---

## 可选：命令行高级模式

```bash
python3 start_one_click.py
```

可追加参数（例如无头模式）：

```bash
python3 start_one_click.py --headless
```
