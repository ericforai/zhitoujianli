🚀 智能投递混合模式测试

测试说明

这个页面用于测试智能投递的混合模式：首次登录使用本地浏览器（有头模式），后续使用后台无头模式。

使用步骤

  1. 点击"启动智能投递"按钮
  2. 如果是首次使用，会下载脚本文件
  3. 在本地安装依赖：npm install playwright ws
  4. 运行脚本：node boss-runner.js
  5. 在本地浏览器中扫码登录Boss直聘
  6. 登录成功后自动切换到无头模式

技术原理

  • 首次登录：下载本地脚本，在用户本地浏览器中显示二维码
  • Cookie保存：登录成功后，Cookie保存在本地
  • 后续使用：使用保存的Cookie，后台无头模式运行
  • WebSocket通信：本地脚本与服务器实时通信
  • 自动切换：登录成功后自动从有头模式切换到无头模式
