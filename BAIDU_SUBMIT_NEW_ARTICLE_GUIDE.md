## 🎯 快速开始（推荐方法）

### 方法一：使用自动提交脚本

#### 步骤1：获取百度Token

1. 访问 **百度站长平台**：https://ziyuan.baidu.com/
2. 使用百度账号登录
3. 进入 **「数据引入」→「链接提交」**
4. 选择 **「主动推送」** 标签
5. 找到接口调用地址（类似）：
   ```
   http://data.zz.baidu.com/urls?site=blog.zhitoujianli.com&token=YOUR_TOKEN_HERE
   ```
6. 复制 `token=` 后面的内容

#### 步骤2：配置提交脚本

编辑脚本文件：

```bash
vi /root/zhitoujianli/scripts/submit-new-article-to-baidu.sh
```

找到这一行并替换Token：

```bash
# 修改前
TOKEN="YOUR_BAIDU_TOKEN"

# 修改后（示例）
TOKEN="wds5zmJ4sTAPlxuN"  # 替换为你的实际Token
```

保存并退出（`:wq`）

#### 步骤3：执行提交

```bash
bash /root/zhitoujianli/scripts/submit-new-article-to-baidu.sh
```

## 🔍 常见问题解答

### Q1: 提交后多久能被收录？

**A**: 时间不确定，取决于多种因素：

| 情况 | 预计时间 |
|-----|---------|
| 网站权重高 | 1-3天 |
| 网站权重一般 | 3-7天 |
| 新网站 | 1-4周 |
| 内容质量差 | 可能不收录 |

**本文章预计**：3-7天（网站已有一定权重）

### Q2: 提交成功但搜索不到？

**A**: 提交成功≠已收录，需要等待：

1. 百度需要时间抓取页面
2. 需要时间分析内容质量
3. 需要时间建立索引

**解决方法**：
- 耐心等待（至少7天）
- 使用 `site:` 命令验证
- 检查robots.txt是否阻止抓取

### Q3: 如何查看是否被百度收录？

**A**: 三种方法：

**方法1：site命令**（最准确）
```
site:blog.zhitoujianli.com/college-graduate-job-hunting-2025/
```

**方法2：完整URL搜索**
```
https://blog.zhitoujianli.com/college-graduate-job-hunting-2025/
```

**方法3：百度站长平台**
- 登录后台
- 查看「数据监控」→「索引量」

### Q4: 每天可以提交多少次？

**A**: 取决于配额：

| 提交方式 | 每日配额 |
|---------|---------|
| 主动推送API | 10,000-100,000（根据网站权重） |
| 手动提交 | 10-20条 |
| Sitemap | 无限制（但不建议频繁更新） |

**建议**：
- 新文章立即主动推送
- 每天批量推送不超过50条
- Sitemap保持每日自动更新

### Q5: Token会过期吗？

**A**: 通常不会过期，但可能发生变化：

**Token失效情况**：
- 网站重新验证
- 站长平台账号变更
- 百度系统升级

**预防措施**：
- 定期检查Token是否有效
- 保存好当前Token
- 测试失败时及时更新

### Q6: 为什么我的配额那么少？

**A**: 配额由网站质量决定：

**影响因素**：
- 网站历史表现
- 内容原创度
- 更新频率
- 用户互动量
- 外部链接质量

**提升方法**：
- 持续发布优质原创内容
- 保持稳定更新频率
- 提升网站整体权重
