# 如何查看UTM参数的具体数据

## 📊 方法1：在百度统计中查看（推荐）

### 步骤1：进入"全部来源"页面

1. 登录 [百度统计](https://tongji.baidu.com/)
2. 选择你的网站
3. 左侧菜单：**来源分析** → **全部来源**

### 步骤2：点击"外部链接"

在数据表格中，找到 **"来源类型"** 列，点击 **"外部链接"** 这一行

### 步骤3：查看详细URL列表

点击后，会进入"外部链接"的详细页面，可以看到：

**数据表格包含：**
- **来源URL**：完整的URL，包括UTM参数
- **浏览量(PV)**：该URL的访问次数
- **访客数(UV)**：访问该URL的独立用户数
- **IP数**：访问该URL的独立IP数
- **跳出率**：该URL的跳出率
- **平均访问时长**：用户在该URL的平均停留时间

**示例数据：**
```
来源URL: https://zhitoujianli.com/register?utm_source=wechat&utm_medium=article&utm_campaign=launch_2025&utm_content=launch_article
浏览量: 25
访客数: 20
IP数: 18
跳出率: 40%
平均访问时长: 00:03:25
```

### 步骤4：筛选和分析UTM参数

#### 筛选特定来源
- 在URL列表中，查找包含 `utm_source=wechat` 的URL
- 可以看到所有来自微信公众号的访问

#### 筛选特定活动
- 查找包含 `utm_campaign=launch_2025` 的URL
- 可以看到所有来自"2025启动活动"的访问

#### 对比不同内容
- 查找包含 `utm_content=launch_article` 的URL
- 对比不同 `utm_content` 值的访问量和转化率

## 📊 方法3：导出数据到Excel分析（推荐用于深度分析）

### 步骤1：导出数据

1. 在"外部链接"页面
2. 查找"导出"或"下载"按钮
3. 导出为Excel或CSV格式

### 步骤2：在Excel中分析

#### 提取UTM参数
可以使用Excel公式提取UTM参数：

**提取utm_source：**
```excel
=MID(A2,FIND("utm_source=",A2)+11,FIND("&",A2,FIND("utm_source=",A2))-FIND("utm_source=",A2)-11)
```

**提取utm_medium：**
```excel
=MID(A2,FIND("utm_medium=",A2)+11,FIND("&",A2,FIND("utm_medium=",A2))-FIND("utm_medium=",A2)-11)
```

**提取utm_campaign：**
```excel
=MID(A2,FIND("utm_campaign=",A2)+13,FIND("&",A2,FIND("utm_campaign=",A2))-FIND("utm_campaign=",A2)-13)
```

#### 创建数据透视表
1. 选择数据范围
2. 插入 → 数据透视表
3. 行：UTM参数（如utm_source）
4. 值：浏览量、访客数、跳出率等
5. 可以快速对比不同UTM参数的效果

## 📊 方法4：使用Google Analytics查看（更强大的分析）

### 步骤1：进入Google Analytics

1. 登录 [Google Analytics](https://analytics.google.com/)
2. 选择你的媒体资源（G-S6YNCB5EDV）

### 步骤2：查看UTM参数报告

#### 方式1：获取报告
1. 左侧菜单：**报告** → **获取** → **流量获取**
2. 查看"来源/媒介"维度
3. 可以看到 `wechat / article` 的访问数据

#### 方式2：探索报告（推荐）
1. 左侧菜单：**探索**
2. 创建新的探索报告
3. 添加维度：
   - `utm_source`（来源）
   - `utm_medium`（媒介）
   - `utm_campaign`（活动）
   - `utm_content`（内容）
4. 添加指标：
   - `会话数`（访问次数）
   - `用户数`（独立用户）
   - `事件数`（注册转化）
   - `转化率`
5. 可以按任意UTM参数筛选和分组

#### 方式3：转化事件报告
1. 左侧菜单：**报告** → **参与度** → **事件**
2. 查找事件：`sign_up`
3. 点击事件名称，查看详细数据
4. 可以看到每个UTM参数的注册转化次数

## 📈 数据分析技巧

### 1. 对比不同来源的效果

**查看数据：**
- `utm_source=wechat` vs `utm_source=zhihu` vs `utm_source=douyin`
- 对比：访问量、转化率、平均访问时长

**分析：**
- 哪个来源带来的访问量最多？
- 哪个来源的转化率最高？
- 哪个来源的用户质量最好？

### 2. 对比不同内容的效果

**查看数据：**
- `utm_content=launch_article` vs `utm_content=menu_register`
- 对比：访问量、转化率

**分析：**
- 哪个内容的点击率更高？
- 哪个内容的转化率更高？
- 应该重点推广哪个内容？

### 3. 分析转化漏斗

**完整转化路径：**
1. 访问注册页面（带UTM参数）
2. 填写注册表单
3. 完成注册（`sign_up` 事件）

**关键指标：**
- **访问注册页面的用户数**（来源分析）
- **完成注册的用户数**（事件报告）
- **注册转化率** = 完成注册 / 访问注册页面

**示例：**
```
utm_source=wechat:
- 访问注册页面: 100人
- 完成注册: 25人
- 注册转化率: 25%

utm_source=zhihu:
- 访问注册页面: 50人
- 完成注册: 15人
- 注册转化率: 30%
```

**结论：** 虽然微信来源的访问量更多，但知乎来源的转化率更高。

## 🎯 关键指标说明

### 访问指标

| 指标 | 说明 | 如何查看 |
|------|------|---------|
| **浏览量(PV)** | 该UTM链接的总访问次数 | 外部链接详情页面 |
| **访客数(UV)** | 访问该UTM链接的独立用户数 | 外部链接详情页面 |
| **IP数** | 访问该UTM链接的独立IP数 | 外部链接详情页面 |

### 质量指标

| 指标 | 说明 | 如何查看 |
|------|------|---------|
| **跳出率** | 只访问一个页面就离开的比例 | 外部链接详情页面 |
| **平均访问时长** | 用户在该页面的平均停留时间 | 外部链接详情页面 |
| **平均访问页数** | 用户平均访问的页面数 | 外部链接详情页面 |

### 转化指标

| 指标 | 说明 | 如何查看 |
|------|------|---------|
| **注册转化数** | 完成注册的用户数 | Google Analytics事件报告 |
| **注册转化率** | 注册转化数 / 访问注册页面数 | 计算得出 |
| **平均转化时间** | 从访问到注册的平均时间 | Google Analytics转化报告 |

## 📊 实际案例分析

### 案例：公众号启动文章效果分析

**UTM链接：**
```
https://zhitoujianli.com/register?utm_source=wechat&utm_medium=article&utm_campaign=launch_2025&utm_content=launch_article
```

**查看步骤：**

1. **百度统计 - 外部链接详情**
   - 找到该URL
   - 查看：浏览量 50，访客数 45，跳出率 30%

2. **Google Analytics - 事件报告**
   - 查找 `sign_up` 事件
   - 筛选 `utm_source=wechat AND utm_campaign=launch_2025`
   - 查看：注册转化数 15

3. **计算转化率**
   - 注册转化率 = 15 / 50 = 30%

4. **对比其他来源**
   - 知乎来源：访问量 30，注册转化 12，转化率 40%
   - **结论：** 知乎来源的转化率更高，但微信来源的访问量更多
