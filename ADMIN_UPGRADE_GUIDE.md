# 管理员手动升级用户套餐 - 操作指南

## 🎯 适用场景

当用户联系客服要求升级套餐时，客服确认用户已支付后，管理员可以在后台手动为用户升级。

---

## 📋 操作步骤（5步完成）

### 步骤1：登录管理后台

```
1. 访问：https://zhitoujianli.com
2. 使用管理员账号登录：admin@zhitoujianli.com
3. 自动跳转到管理后台
```

---

### 步骤2：进入用户管理

```
1. 点击左侧菜单"用户管理"
2. 查看所有用户列表
```

---

### 步骤3：找到目标用户

在用户列表中找到要升级的用户，查看其当前套餐类型：
- **FREE** = 求职入门版（免费）
- **BASIC** = 高效求职版（¥49/月）
- **PROFESSIONAL** = 极速上岸版（¥99/月）

---

### 步骤4：点击"升级套餐"按钮

1. 点击用户行的 **蓝色"升级套餐"** 按钮
2. 弹出对话框：
```
请选择目标套餐：

1. 求职入门版（免费） ← 当前套餐
2. 高效求职版（¥49/月）
3. 极速上岸版（¥99/月）

请输入数字（1-3）：
```

3. 输入数字：
   - `1` = 免费版
   - `2` = 高效版
   - `3` = 极速版

---

### 步骤5：确认升级

1. 系统二次确认：
```
确定要将用户 user@example.com 的套餐从
求职入门版（免费） 改为 高效求职版（¥49/月） 吗？
```

2. 点击"确定"
3. 系统提示："套餐升级成功！"
4. 用户列表自动刷新，显示新套餐类型

---

## ✅ 升级后的效果

### 对于用户

**立即生效**：
- 用户刷新页面后，套餐立即更新
- 配额自动更新为新套餐的配额
- 可以使用新套餐的所有功能

**在Dashboard可见**：
- 套餐名称变更
- 配额限制更新
- 可使用次数增加

---

## 📊 三个版本的配额对比

| 功能 | 免费版 | 高效版 | 极速版 |
|------|--------|--------|--------|
| 简历基础优化 | 1次 | 不限次 | 不限次 |
| 简历高级优化 | 0次 | 1次 | 3次 |
| 每日投递 | 5次/天 | 30次/天 | 100次/天 |

## 🔧 常见场景处理

### 场景1：用户支付了高效版（¥49）

**客服操作**：
1. 确认用户已支付¥49
2. 登录管理后台
3. 找到用户
4. 点击"升级套餐"
5. 选择"2"（高效求职版）
6. 确认升级
7. 通知用户："您的套餐已升级，请刷新页面查看"

---

### 场景2：用户支付了极速版（¥99）

**客服操作**：
1. 确认用户已支付¥99
2. 登录管理后台
3. 找到用户
4. 点击"升级套餐"
5. 选择"3"（极速上岸版）
6. 确认升级
7. 通知用户："您的套餐已升级，请刷新页面查看"

---

### 场景3：用户要求降级/退款

**客服操作**：
1. 确认退款已处理
2. 登录管理后台
3. 找到用户
4. 点击"升级套餐"
5. 选择"1"（求职入门版/免费）
6. 确认升级
7. 用户恢复免费版配额

---

## 🚨 注意事项

### ⚠️ 升级前必须确认

1. **确认用户已支付**：绝对不能未支付就升级
2. **确认支付金额**：¥49对应高效版，¥99对应极速版
3. **确认用户邮箱**：确保升级正确的用户

### ⚠️ 升级后无法撤销

- 套餐升级后立即生效
- 如需撤销，只能再次手动改回原套餐
- 建议保留操作记录

### ⚠️ 配额使用情况

- 升级套餐**不会清零**已使用的配额
- 只会提高配额上限
- 例如：用户已投递3次 → 升级到高效版 → 变成3/30

---

## 🔍 如何验证升级成功

### 管理员验证

1. 用户列表中套餐类型列显示新套餐
2. 刷新页面确认数据已保存

### 用户验证

1. 用户刷新页面（Ctrl + F5）
2. 访问/dashboard
3. 查看配额显示卡片：
   - 套餐名称已更新
   - 配额限制已更新
   - 可使用次数增加
